## Installation

1. Download this theme
2. Extract to your Typecho `usr/themes/` directory
3. Rename the folder to `fuwari-typecho` 
4. Login to your Typecho admin panel
5. Go to "Appearance" -> "Themes"
6. Enable the Fuwari theme
